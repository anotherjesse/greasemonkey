Greasemonkey 0.2.5 - Beta

Archived location:
http://downloads.mozdev.org/greasemonkey/greasemonkey_0.2.5.xpi

Changes

  • Introduced GM_xmlhttpRequest, a function script authors can use to make xmlhttp requests across domains.
  • Introduced GM_registerMenuCommand, allowing script authors to add menu items to the Firefox tools menu.
  • Added enable/disable all buttons in the manage screen.
  • Added xhtml doctype support.
